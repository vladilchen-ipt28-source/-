# Лабораторна робота №6: Модульний додаток OpenCV на C++

Програма захоплює відеопотік з камери, виводить його у вікно та дозволяє накладати різні фільтри обробки зображень у реальному часі за допомогою гарячих клавіш.

## Вимоги до системи
* **ОС:** Linux (тестовано на Ubuntu 24.04 / 22.04 LTS)
* **Залежності:** OpenCV (версія 4.6.0 або новіша), CMake (>= 3.10), компилятор G++ (з підтримкою C++17).
* **Залізо:** Наявність веб-камери (підтримка V4L2 бекенду) або відеофайлу для fallback-режиму.

## Інструкція з білду та запуску
Для встановлення залежностей, збірки та старту програми виконайте по черзі команди в терміналі:

1. Встановлення необхідних пакетів:
```bash
   ./preinstall.sh
```
2.Компіляція проєкту:

```bash
./build.sh
```
3. Запуск програми:

```bash
./run.sh
```
4. Керування програмою (Гарячі клавіші)
   
1 — Оригінальне відео

2 — Чорно-білий режим (Grayscale)

3 — Інверсія кольорів (Negative)

4 — Виявлення контурів (Canny)

5 — Розмиття (Gaussian Blur)

6 — Бінаризація (Threshold)

+ або = — Збільшити яскравість кадру

- — Зменшити яскравість кадру

S / s — Зробити скріншот поточного кадру (зберігається в папку проєкту)

Esc — Вихід з програми
