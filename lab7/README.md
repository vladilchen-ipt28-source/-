# Лабораторна робота №7: Computer Vision та багатопотоковість у C++

## Опис проєкту
Проєкт реалізує модульний додаток для детекції облич на відеопотоці в реальному часі за допомогою модуля `cv::dnn` бібліотеки OpenCV та претренованої нейромережі ResNet-10 (SSD). 

Для забезпечення стабільної частоти кадрів (FPS) та запобігання блокуванню графічного інтерфейсу важкі обчислення нейромережі винесені в окремий фоновий потік за допомогою `std::thread`, а обмін даними між потоками безпечно синхронізовано через `std::mutex` та механізм захисту `std::lock_guard`.

## Опис архітектури проєкту
Програма побудована за модульним принципом і складається з таких компонентів:
- **CameraProvider:** відповідає за зчитування кадрів з відеофайлу (`video.mp4`).
- **KeyProcessor:** обробляє натискання гарячих клавіш та зберігає поточний режим роботи програми через `enum`.
- **FrameProcessor:** виконує класичну обробку зображення (фільтри, накладання тексту FPS та лічильника кадрів).
- **Display:** відповідає за створення графічного вікна ОС, роботу інтерактивного слайдера яскравості (Trackbar) та виведення результату на екран (`imshow`).
- **FaceDetector:** керує фоновим потоком воркера (`std::thread`), ініціалізує нейромережу Caffe та асинхронно проводить інференс, захищаючи спільні координати знайдених облич за допомогою `std::mutex`.

## Вимоги до системи
- **ОС:** Linux (тестовано на Ubuntu 24.04 LTS на VMware)
- **Компілятор:** GCC / G++ з підтримкою стандарту C++11 або вище
- **Інструменти збірки:** CMake (версія 3.10 або новіша), GNU Make
- **Залежності:** OpenCV (версія 4.6.0 або новіша) з підтримкою модуля DNN та системна бібліотека потоків `Threads`

## Інструкція з використання скриптів автоматизації
Усі скрипти автоматизації є виконавчими. Для повного циклу підготовки, збірки та запуску проєкту виконайте по черзі команди в терміналі всередині папки проєкту:

1. Встановлення залежностей та автоматичне завантаження архітектури й ваг нейромережі з GitHub:
   ```bash
   ./preinstall.sh
   ```
2.Компіляція проєкту за допомогою CMake:

  ```bash
./build.sh
  ```
3.Запуск готового бінарного файлу:

  ```bash
./run.sh
  ```
Керування програмою (Гарячі клавіші)
0 — Оригінальне відео.

1 — Чорно-білий фільтр (Grayscale).

2 — Фільтр контурів Кенні (Canny Edge).

3 — Розмиття Гаусса (Gaussian Blur).

4 — Інверсія кольорів (Bitwise Not).

5 — Кольоровий ефект зсуву каналів (Glitch effect).

F / f — Увімкнення багатопотокової детекції облич (Face Detection).

Слайдер "Brightness" — Зміна яскравості в реальному часі.

Клік ЛКМ по відео — Логування координат миші в консоль.

Esc або q — Завершення роботи програми.
