# Лабораторна робота №7: Computer Vision та багатопотоковість у C++ (Рівень 1)

Проєкт реалізує модульний додаток для детекції облич на відеопотоці в реальному часі за допомогою модуля `cv::dnn` бібліотеки OpenCV та претренованої нейромережі ResNet-10 (SSD).

## Вимоги до системи
* **ОС:** Linux (тестовано на Ubuntu 22.04 / 24.04 LTS)
* **Компілятор:** GCC з підтримкою стандарту C++17
* **Інструменти збірки:** CMake (версія 3.10 або новіша)
* **Залежності:** OpenCV (версія 4.6.0 або новіша)

## Інструкція з використання скриптів автоматизації

Усі скрипти автоматизації є виконавчими. Для повного циклу підготовки, збірки та запуску проєкту виконайте по черзі команди в терміналі:

1. Встановлення залежностей та завантаження моделей нейромережі:
   ```bash
   ./preinstall.sh
    ```
2. Компіляція проєкту:

 ```bash
./build.sh
 ```
3. Запуск готового бінарного файлу:

  ```bash
./run.sh
 ```
4. Керування програмою (Гарячі клавіші)
   
1–6 — Фільтри обробки зображення з попередньої лабораторної роботи.

7 — Увімкнення режиму детекції облич (Face Detection).

Esc — Вихід з програми.
