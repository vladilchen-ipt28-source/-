# Лабораторна робота №5
## Наука про дані: обмін результатами та початковий аналіз

### Опис завдання
**Мета роботи:** Реалізація веб-додатка на базі бібліотеки Streamlit для інтерактивного аналізу та візуалізації часових рядів рослинних індексів (VCI, TCI, VHI) по областях України (на основі датасету з Лабораторної роботи №2).

**Елементи керування (Sidebar):**
* Dropdown для вибору аналізованого індексу (VCI / TCI / VHI).
* Dropdown для вибору конкретної області України.
* Слайдери для фільтрації часових інтервалів (діапазон тижнів та діапазон років).
* Чекбокси для сортування даних за зростанням або спаданням з інтегрованою логікою попередження при конфлікті вибору.
* Кнопка Reset для миттєвого скидання всіх фільтрів до початкового стану через механізм `st.session_state`.

**Архітектура інтерфейсу:** Програму розділено на дві функціональні зони: інтерактивні елементи керування винесено в бічну панель (`st.sidebar`), а результати (таблиці та графіки) відображаються в основній зоні, розбитій на три тематичні вкладки (`st.tabs`).

---

### Системні вимоги
* **ОС:** Linux (Ubuntu 20.04/22.04/24.04) або будь-яка сумісна Unix-система.
* **Інтерпретатор:** Python версії 3.10 або вище.
* **Необхідні пакети:** `streamlit`, `pandas`, `numpy`, `matplotlib` (встановлюються автоматично).

---

### Інструкція з налаштування середовища та запуску

Для перевірки роботи програми на чистій системі у повному обсязі автоматизовано процеси створення оточення та завантаження даних. Виконайте три прості кроки в терміналі:

#### 1. Попереднє встановлення та автоматичне викачування даних
Скрипт автоматично оновить системні пакети, створить ізольоване віртуальне середовище `venv`, оновить менеджер `pip`, встановить залежності з `requirements.txt` та завантажить повний набір актуальних CSV-файлів рослинних індексів від NOAA в директорію `vhi_data`:
```bash
./preinstall.sh
```
#### 2. Запуск локального вебсервера
Скрипт автоматично активує створене віртуальне середовище venv та ініціює старт Streamlit-сервера:

```bash
./run.sh
```
#### 3. Робота з додатком
Після виконання скрипту запуску у вашому браузері автоматично відкриється сторінка за адресою http://localhost:8501 (або http://127.0.0.1:8501) з готовим інтерактивним веб-інтерфейсом для аналізу даних.
