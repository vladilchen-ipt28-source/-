# Лабораторна робота №2: Наука про дані — підготовчий етап

## Опис завдання

### Частина 1: Моніторинг індексу VHI
* Організація автоматичного завантаження структурованих текстових файлів з сервера NOAA за допомогою модуля `urllib`.
* Реалізація механізмів запобігання повторного завантаження даних та уникнення колізій (додавання міток часу до файлів).
* Очищення та первинна обробка даних (Data Cleaning) засобами бібліотеки `pandas`.
* Переіндексація адміністративних одиниць України відповідно до українського алфавіту.
* Створення функцій для формування аналітичних вибірок та розрахунку статистичних показників (мінімум, максимум, середнє, медіана).

### Частина 2: Аналіз енергоспоживання (Individual Household Electric Power Consumption)
* Очищення та обробка великого датасету (понад 2 млн записів), обробка пропущених значень (`?` -> `NaN`).
* Формування складних умовних вибірок та покрокове крокове зрізання (`iloc`).
* Векторна нормалізація (Min-Max Scaling) та стандартизація (Z-score Normalization) ознак.
* Обчислення коефіцієнтів лінійної та рангової кореляції Пірсона і Спірмена.
* Застосування техніки One Hot Encoding для категоріальних ознак.
* Профілювання часових витрат на виконання кожної процедури за допомогою модуля `timeit`.

## Інструкція з налаштування середовища та запуску

### 1. Налаштування віртуального середовища
Перейдіть у терміналі до теки проєкту та виконайте команди для створення й активації якщо ви перебуваєте в корені репозиторію:

```bash
# Створення віртуального середовища
python -m venv venv

# Активація (для Windows):
venv\Scripts\activate

# Активація (для Linux / MacOS):
source venv/bin/activate
```

### 2. Встановлення залежностей
Встановіть необхідні для роботи бібліотеки, що зафіксовані у локальному файлі конфігурації:
```bash
pip install -r requirements.txt
```
### 3. Інструкція щодо отримання датасетів
Дані Частини 1 (VHI): Завантажуються автоматично під час виконання комірок у блокноті lab2_1.ipynb. Скрипт самостійно створює теку vhi_data та обходить SSL-блокування сервера.

Дані Частини 2 (Енергоспоживання): Необхідно завантажити архів датасету Individual Household Electric Power Consumption з репозиторію UCI, розпакувати його та покласти файл household_power_consumption.txt у теку lab2/ поруч із файлом блокнота.

### 4. Запуск скриптів
Для запуску та перегляду результатів відкрийте середовище Jupyter Notebook або скористайтеся розширенням Jupyter у VS Code:

Виконайте послідовний запуск комірок у lab2_1.ipynb для перегляду першої частини.
```bash
jupyter notebook lab2_1.ipynb
jupyter notebook lab2_2.ipynb
```
Виконайте послідовний запуск комірок у lab2_2.ipynb для перегляду професійної аналітики другої частини (із виведенням результатів часового профілювання процедур).
