# Лабораторна робота №4: Візуалізація даних 2

## Опис завдання
* **Мета роботи:** Реалізація інтерактивної програми на базі бібліотеки `matplotlib` для моделювання гармонічних сигналів виду $y(t) = A \cdot \sin(\omega t + \phi)$ із накладанням випадкового шуму та оцінки цифрової фільтрації.
* **Функція:** `harmonic_with_noise(amplitude, frequency, phase, noise_mean, noise_covariance, show_noise)`.
* **Фільтрація:** Застосування низькочастотного цифрового фільтра Баттерворта з бібліотеки `scipy.signal` для згладжування сигналу.
* **Інтерактивні елементи:**
  * Слайдери регулювання параметрів гармоніки (Amplitude, Frequency, Phase) та характеристик Гаусового шуму (Noise Mean, Noise Covariance).
  * Слайдер Cutoff Frequency для керування частотою зрізу фільтра.
  * Чекбокси незалежного перемикання режимів видимості графіків.
  * Кнопка автоматичного скидання (`Reset`) параметрів до вихідного стану.
* **Архітектурна вимога:** Забезпечено повну ізоляцію шуму від параметрів гармоніки (зміна амплітуди чи частоти не перегенеровує випадкові коливання наново).

## Системні вимоги
* **Інтерпретатор:** Python версії `3.10` або вище.
* **Необхідні пакети:** `numpy`, `matplotlib`, `scipy`, `PyQt5`.

## Інструкція з налаштування середовища та запуску

### 1. Створення та активація віртуального середовища
Відкрийте термінал та переконайтеся, що перебуваєте всередині папки `lab4`. Виконайте команди:
```bash
# Створення venv
python -m venv venv

# Активація (для Windows):
.\venv\Scripts\Activate.ps1
```
### 2. Встановлення залежностей
Встановіть необхідні бібліотеки обробки сигналів та графічний рушій Qt через локальний файл конфігурації:

```bash
pip install -r requirements.txt
```
### 3. Запуск блокнота
Відкрийте Jupyter Notebook у VS Code, перемкніть ядро (Kernel) на створений локальний lab4/venv та виконайте послідовний запуск комірок. Програма автоматично ініціалізує плагіни Qt та відкриє окреме інтерактивне вікно керування.
